# VIDMARK STUDIO App Design

This document governs the local macOS app UI.

## Product Feel

VIDMARK STUDIO should feel like a quiet premium production console: dark, minimal, focused, and practical.

Use:

- System dark mode by default
- Clean spacing and restrained hierarchy
- Small accent color only for state and action
- Native macOS controls where possible
- Clear timecode and review-mark interactions
- Minimal surfaces with no decorative clutter

Avoid:

- White or light root backgrounds
- Loud gradients
- Decorative chrome that competes with the video
- Full editor-suite complexity
- Provider-specific account language in the core app

## Review UI

The review interface should make it easy to:

- Play the current master
- See exact timecode
- Add a mark at the current time
- Pick the problem type
- Pick the requested fix
- Leave a short note
- Export marks for editors or automation agents

The app should stay simple enough that a creator can review a video without feeling like they are operating a full non-linear editor.
