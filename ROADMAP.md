# Roadmap

VIDMARK STUDIO is intentionally practical and narrow. The roadmap favors features that help creators review and ship video assets faster without becoming a full editing suite.

## Near Term

- Add sample project manifests that use placeholder media paths only.
- Improve review mark filtering and grouping.
- Add keyboard shortcuts for mark creation and transport controls.
- Add lightweight project status indicators.
- Add export presets for common vertical and square formats.

## Mid Term

- Add optional contact sheet generation for review sessions.
- Add automated silent/audio-present detection in the UI.
- Add basic scene-boundary visualization for long masters.
- Improve smart reframing confidence scoring.
- Add a small test suite for Python helper engines.

## Later

- Add optional plugin points for external creator pipelines.
- Add project templates for common video-production workflows.
- Add batch review-package export.
- Add richer accessibility and localization support.

## Non-Goals

- Full timeline editing
- Cloud account management
- Bundled paid-service connectors
- Provider-specific generation workflows in the core app
